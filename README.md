# Night Owl Hub

Fine dining under the stars. A restaurant website with atmospheric design, menu, gallery, reservations, and contact features.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Scripts

- `npm run dev` — Start dev server with hot reload
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint
- `npm run test` — Run tests
